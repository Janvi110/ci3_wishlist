App Installation

Installing easy wishlist app is easy and quick.

If you already had the Shopify site and want to install App, you need to just follow the instructions mentioned below:

1. In your back office go to: Apps:

  • Click on: Visit Shopify App Store

2. Go to App store:

  • Search on app store: Easy Wishlist

  • Click on: Add app

  • Click on: Install App
